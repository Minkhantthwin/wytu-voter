# Candidate Photos

Place candidate photos in this directory with the following naming convention:

## For Seeded Candidates:
- `king1.jpg` - King Candidate 1
- `king2.jpg` - King Candidate 2
- `king3.jpg` - King Candidate 3
- `queen1.jpg` - Queen Candidate 1
- `queen2.jpg` - Queen Candidate 2
- `queen3.jpg` - Queen Candidate 3

## Recommended Image Specs:
- Format: JPG, PNG, or WEBP
- Size: 500x500px (square)
- Max file size: 2MB
- Aspect ratio: 1:1 (square)

Photos will be served at: `http://localhost:3000/uploads/candidates/filename.jpg`
